二、第二章

基础数据类型
int：整数类型
float：小数，浮点数（数据范围是无限的）
bool：用来做条件判断的 取值范围：Ture，False 在pytho中0是False其余是Ture，非空字符串为Ture，空字符串为False

字符串
1.字符串格式化 %s字符串占位 %d整数占位 f-string格式化
2.索引和切片 索引：按照位置提取元素 s[start:end]从start切到end，end不取，只能从左往右切 s[::-1]表示从右往左，-1表示步长
3.字符串常规操作，不会改变原字符串 s.upper把所有字母变成大写
4.替换和切割 字符串.strip去掉左右两边的空白符（空格，\t,\n）.replace(“ “ “”)去掉所有空格。 .split(切割内容) 切割
5.查找和判断 in可以做条件上的判断 .startwith/.endwith以开头和结尾 isdigit判断字符串是否由整数组成

列表
list
a = [“asdd”,”dad”[1,2,就hi的]]
像字符串一样有索引和切片
lst.append添加内容 .insert插入 .extend合并两个表 .remove直接删除元素
lst.sort()对列表升序 lst.sort(reverse=True)降序排列 for循环的bug

元组
tuple ,固定元组元素后地址不能改，如果元组中只有一个元素要加逗号才能认为是元组

集合
set 特点无序 s = {}插入元素才能是集合 存储数据需是可哈希 可哈希类型(不可变的数据类型)：int str tuple bool 可变的数据类型：list，dict，set
s = set（）创建空集合 &交集 |并集 -差集 set集合可以去除重复

字典
dict
字典是以键值对的形式进行存储数据的
{key：value，key2：value} 字典的key必须是可哈希的，key值不可重复，有新的value值进来直接覆盖原来的值
dic.setdefaut()设置默认值 dic.pop(key)根据key删除
print(dic[key])如果key不存在，程序报错，适用于key没问题
print（dic.get(key)）key不存在，返回none，适用于不确定key .keys()拿所有key值 .values()拿所有value值
for key.value in dic.items()拿到所有key和value值

第四章

函数
定义：对某一个特定的功能或者代码块进行封装，在需要使用该功能的时候直接调用即可
def 函数的名字():
被封装的功能或者代码块
调用：函数的名字（）
参数：在函数调用时给函数传递一些信息，分类：1.形参：在函数定义时，需要准备一些变量来接收信息
位置参数：按照位置一个一个的去声明变量
默认值参数：在函数声明时给变量一个默认值
动态传参：表示位置参数的动态传参，接受的参数放在元组里
*表示接受关键字的动态传参，接受的参数放在字典里
位置参数在前面，，默认值在后面，* 参数可以随意搭配
2.实参：实际在调用的时候传递的信息
位置参数：按照位置进行传递参数
关键字参数：按照参数的名字进行传递参数
混合参数：位置参数放在前面，关键字参数放在后面
*把列表打散成位置参数进行传递
**把字典自动转换成关键字参数进行传递
返回值：函数执行之后，会给调用方一个结果
def fuc(a,b)

print(a)
关于return： 1.如果函数内没有return，此时外界收到的是None
2.如果写了return
1.只写了return，后面不跟数据，此时收到的依然是None
2.return 值，此时表示函数有一个返回值 外界收到一个值
3.return 值1 值 2…外界可以收到多个值
函数如果执行到了return函数就会立即停止返回内容,函数内的return的后续代码不会执行
