1.创建vue项目

  • 新建一个文件夹，在vscode中下载vue的插件并在vscode命令终端下输入npm create vue@latest,根据弹出的命令行输入命令

  • 打开创建的vue项目，在该项目的终端下输入npm install element-plus --save 、npm install -D unplugin-vue-components unplugin-auto-import

  • 修改main.js、App.vue、vite.config.js文件修改内容如下：

    main.js

    App.vue

    vite.config.js

  • 在App.vue文件中进行改写
